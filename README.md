This project is an IoT-based automatic fish feeder using ESP32.

## Features
- Automatic feeding at fixed times
- Manual feeding via Blynk app
- Servo-controlled food dispensing
- Buzzer alert during feeding
- Real-time clock using NTP
- Mobile notification after feeding

## Hardware Used
- ESP32
- Servo motor
- Buzzer
- Fish food container

## Software Used
- Arduino IDE
- Blynk IoT
- ESP32Servo Library
